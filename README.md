# 🛒 Amazon Clone

A responsive Amazon-inspired e-commerce web application built using HTML, CSS, and JavaScript. This project recreates the core user interface and shopping experience of Amazon, including product browsing, navigation, search functionality, shopping cart interactions, and modern responsive design.

---

# 📖 Overview

Amazon Clone is a frontend web development project designed to replicate the look and feel of the Amazon marketplace. The application focuses on creating a professional e-commerce user interface with responsive layouts, product displays, category sections, and interactive components.

The project demonstrates practical frontend development skills, responsive design principles, DOM manipulation, and modern web UI implementation.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/AbhishekVijay2907/AMAZON_CLONE.git
```

## Open Project

```bash
cd AMAZON_CLONE
```

Run the project by opening:

```text
index.html
```

in your browser.
